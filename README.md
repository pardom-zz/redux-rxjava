# Redux RxJava
RxJava extentions for Redux for Kotlin

Download
--------

```groovy
repositories {
	...
	maven { url "https://jitpack.io" }
}
```

```groovy
// RxJava 1
compile 'com.github.pardom.redux-rxjava:redux-rxjava:<version>'

// RxJava 2
compile 'com.github.pardom.redux-rxjava:redux-rxjava2:<version>'
```
