# Redux RxJava
RxJava extentions for Redux for Kotlin

Download
--------

```groovy
repositories {
	...
	maven { url "https://jitpack.io" }
}
```

```groovy
// RxJava 1
compile 'com.github.pardom.redux-rxjava-kotlin:rxjava:<version>'
// RxJava 2
compile 'com.github.pardom.redux-rxjava-kotlin:rxjava2:<version>'
```
